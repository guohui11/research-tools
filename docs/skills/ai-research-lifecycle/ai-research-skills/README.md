# AI-Research-SKILLs

分类：AI 研究全流程 Skills

上游仓库：[Orchestra-Research/AI-research-SKILLs](https://github.com/Orchestra-Research/AI-research-SKILLs)

## 简介

围绕 AI 研究生命周期构建的技能集合，覆盖文献综述、想法生成、实验设计、训练评估、结果分析和论文写作。

## 研究生使用建议

做机器学习、深度学习、LLM、Agent 或数据科学课题的研究生。

按研究阶段选择技能：先综述与选题，再实验与复现，最后分析、绘图和写作。
